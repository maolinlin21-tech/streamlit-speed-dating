Project Title:
Exploratory Analysis of Speed Dating Outcomes

Description:
This project explores factors associated with dating outcomes using a real-world
speed dating dataset. The analysis focuses on perceived attractiveness, shared
interests, liking decisions, and match outcomes.

Dataset:
Speed Dating Dataset (Kaggle)
https://www.kaggle.com/datasets/ulrikthygepedersen/speed-dating

Files:
- analysis.ipynb: Jupyter Notebook containing data cleaning, analysis, and visualizations
- speeddating.csv : Original dataset file
- README.txt : Project description and instructions

How to Run:
1. Download the dataset from the link above.
2. Place the file "speeddating.csv" in the same folder as the Jupyter Notebook.
3. Open "analysis.ipynb" and run all cells from top to bottom.
